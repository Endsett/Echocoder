# EchoCoder Wiki

Welcome to the **EchoCoder** documentation — the AI-native IDE extension for VS Code powered by OpenClaude.

EchoCoder transforms Visual Studio Code into a fully autonomous coding agent with Plan → Approve → Execute → Verify workflows, inline edits, ghost text completions, and multi-file composer mode.
